## TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install nodejs
> pkg install libwebp
> pkg install git
> pkg install ffmpeg
> pkg install yarn
> termux-setup-storage
> git clone https://github.com/Lexxy24/NEOBOTV4.git
> cd NEOBOTV4
> yarn
> rm -r session.json
> npm start
```
